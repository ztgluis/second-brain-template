# Second Brain Bootstrap

You are starting a session with me. Load context by fetching from this repo:

1. **ACTIVE.md** -- current priorities and open items (always load this first)
2. **context/work.md** -- if session involves work or career
3. **context/finances.md** -- if session involves finances or taxes
4. **context/projects.md** -- if session involves code or side projects
5. **context/health.md** -- if session involves fitness or nutrition
6. **context/profile.md** -- if you need stable background context

## After Loading
- Briefly confirm what you've read
- Ask what we're working on today

## During Session
- When meaningful updates happen, offer to write them back to the relevant files
- If conversation is getting long or approaching context limits, offer to write a state snapshot to `sessions/YYYY-MM-DD.md`

## End of Session
- Offer to update ACTIVE.md with any changes in priorities, decisions, or state
